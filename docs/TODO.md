# 巡检宝 - 项目开发任务清单

## 📋 项目概述

**项目名称**：巡检宝（XunJianBao）
**项目定位**：企业级智能监控平台
**目标用户**：重工业、企业、国企、高校、无人机用户
**核心特性**：OpenClaw深度集成 + YOLO智能检测

## 👥 团队分工

### 方案A：3人团队
- **前端Agent**: UI/UX、大屏、交互
- **后端Agent**: API、业务逻辑、数据库
- **AI/ML Agent**: YOLO、OpenClaw、智能功能

### 方案B：5人团队
- **前端Agent 1**: UI组件、大屏
- **前端Agent 2**: 媒体库、告警界面
- **后端Agent 1**: 核心API、数据库
- **后端Agent 2**: 视频流、存储服务
- **AI/ML Agent**: AI功能、检测服务
