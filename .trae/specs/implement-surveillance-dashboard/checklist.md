# 监控大屏模块验收清单

## 基础功能验收

- [ ] 监控大屏页面可以正常访问（`/monitor` 路由）
- [ ] 页面采用深色主题，符合监控场景的专业感
- [ ] 顶部工具栏包含：布局切换按钮、AI助手按钮、状态统计信息
- [ ] 主内容区可以正常显示视频流网格
- [ ] 侧边栏可以正常显示视频流列表

## 多画面布局验收

- [ ] 布局选择器可以正常显示（1x1、2x2、3x3、4x4、5x5选项）
- [ ] 点击布局按钮可以立即切换布局
- [ ] 布局切换时视频窗口平滑过渡，无闪烁
- [ ] 切换布局后，正在播放的视频流保持播放状态
- [ ] 按数字键1-5可以快速切换布局
- [ ] 当前选中的布局按钮有高亮显示

## 视频窗口验收

- [ ] 每个视频窗口有清晰的边框和标题栏
- [ ] 窗口标题栏显示视频流名称
- [ ] 窗口右上角显示状态指示器（绿色=正常、黄色=警告、红色=异常）
- [ ] 鼠标悬停窗口时显示控制按钮（全屏、截图、设置等）
- [ ] 鼠标悬停窗口时显示详细状态信息（帧率、码率、延迟）
- [ ] 双击窗口可以全屏显示该视频
- [ ] 全屏状态下再次双击或按ESC可以退出全屏
- [ ] 点击窗口时，窗口边框高亮显示为焦点窗口

## 无人机专区验收

- [ ] 无人机专区固定显示在页面左侧或顶部
- [ ] 无人机专区包含6个视频窗口（2x3或3x2布局）
- [ ] 每个无人机窗口显示设备名称
- [ ] 每个无人机窗口显示电量百分比
- [ ] 每个无人机窗口显示信号强度图标
- [ ] 每个无人机窗口显示飞行高度
- [ ] 无人机视频流断开时显示"信号丢失"提示
- [ ] 无人机视频流断开后自动尝试重连
- [ ] 点击无人机窗口可以放大到主视频区

## 视频流管理验收

- [ ] 视频流列表可以正常加载和显示
- [ ] 视频流按照优先级排序（在线 > 离线 > 故障）
- [ ] 可以拖拽视频窗口到另一个位置进行交换
- [ ] 拖拽时有半透明预览效果
- [ ] 拖拽时目标位置有高亮提示
- [ ] 拖拽完成后视频流顺序被保存
- [ ] 刷新页面后，视频流顺序保持不变

## 视频播放器验收

- [ ] 视频可以正常播放（WebRTC或HLS）
- [ ] 播放器有加载中状态提示
- [ ] 播放器有错误状态提示
- [ ] 播放器控制条包含：播放/暂停、音量、全屏、截图按钮
- [ ] 点击播放/暂停按钮可以控制视频播放
- [ ] 点击音量按钮可以调节音量或静音
- [ ] 点击全屏按钮可以全屏显示
- [ ] 点击截图按钮可以截取当前画面
- [ ] 播放器实时显示帧率、码率、延迟信息

## 状态监控验收

- [ ] 状态栏显示当前在线视频流数量
- [ ] 状态栏显示当前离线视频流数量
- [ ] 状态栏显示当前告警数量
- [ ] 视频流状态实时更新（WebSocket或轮询）
- [ ] 视频流异常时，窗口边框变为黄色或红色
- [ ] 视频流异常时，窗口右上角显示异常类型图标
- [ ] 鼠标悬停异常图标显示详细信息
- [ ] 有新的告警时，弹出告警提示
- [ ] 告警提示包含告警类型、位置、时间等信息

## AI助手验收

- [ ] 右下角有AI助手悬浮按钮
- [ ] 点击AI助手按钮打开AI面板
- [ ] 按Ctrl+K可以打开AI面板
- [ ] AI面板从右侧滑入，有平滑动画
- [ ] AI面板包含消息列表、输入框、发送按钮
- [ ] 可以输入问题并发送给AI
- [ ] AI响应以流式方式显示
- [ ] AI调用工具时显示工具调用过程
- [ ] AI返回结果后，相关视频流被高亮显示
- [ ] 有快捷问题建议（如"显示所有离线摄像头"）
- [ ] 可以查看历史对话记录
- [ ] 点击面板外部或按ESC可以关闭面板

## 智能监控验收

- [ ] 可以询问"显示所有离线摄像头"并得到正确结果
- [ ] 可以询问"分析最近1小时的告警趋势"并得到分析报告
- [ ] YOLO检测到火灾时，AI自动分析告警严重程度
- [ ] AI分析结果会影响视频流的显示优先级
- [ ] AI生成的告警分析报告显示在AI面板中
- [ ] 可以使用语音控制（如"切换到4画面布局"）
- [ ] 语音指令被正确识别并执行
- [ ] 执行结果显示在AI面板中

## 自定义布局验收

- [ ] 可以进入布局编辑模式
- [ ] 编辑模式下可以拖拽窗口边框调整大小
- [ ] 编辑模式下可以拖拽窗口标题栏调整位置
- [ ] 调整后的布局可以保存
- [ ] 保存的布局可以加载
- [ ] 可以创建多个自定义布局模板
- [ ] 可以删除自定义布局模板
- [ ] 可以重命名自定义布局模板

## 性能优化验收

- [ ] 同时播放9路以下视频流时，所有窗口保持高清
- [ ] 同时播放超过9路视频流时，非焦点窗口自动降低码率
- [ ] 焦点窗口始终保持高清画质
- [ ] 点击窗口时，该窗口自动成为焦点窗口
- [ ] 滚动视频流列表时，只渲染可见区域的窗口
- [ ] 离开视野的窗口自动暂停播放
- [ ] 重新进入视野的窗口自动恢复播放
- [ ] 20路视频流同时播放时，CPU使用率 < 80%
- [ ] 20路视频流同时播放时，内存使用 < 2GB
- [ ] 布局切换时间 < 500ms
- [ ] AI对话响应时间 < 3秒

## 数据缓存验收

- [ ] 视频流列表被缓存，刷新页面无需重新加载
- [ ] 布局配置被缓存，刷新页面后保持不变
- [ ] 视频流预加载功能正常工作
- [ ] 截图被缓存到IndexedDB
- [ ] 缓存数据可以手动清除

- [ ] 视频流播放失败时显示错误提示

- [ ] 网络断开时显示离线提示
- [ ] 网络恢复后自动重连
- [ ] 所有错误都有明确的错误信息
- [ ] 所有错误都可以被用户关闭
