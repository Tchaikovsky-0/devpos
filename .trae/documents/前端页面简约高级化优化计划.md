# 前端页面简约高级化优化计划

## 背景
当前前端页面视觉上过于冗余，每页功能模块分割太强，缺少简约高级感。

## 问题分析

### 1. Center 页面（中控台）
- **问题**：多层嵌套的 WorkspacePanel，过多的 SectionHeader
- **具体表现**：
  - ContextActionStrip → WorkspacePanel → 内部又有多个 SectionHeader
  - 侧边栏列表区域有独立的边框和标题
  - 右侧详情区域分割成多个独立卡片
  - 每个功能区块都有明确的标题和描述文字

### 2. Gallery 页面（图库）
- **问题**：右侧三个 SectionBlock 堆叠，视觉分割过强
- **具体表现**：
  - "当前焦点"、"批处理动作"、"资产概况" 三个区块各自独立
  - 每个区块都有标题 + 描述 + 边框
  - 整体布局显得零散

### 3. Media 页面（媒体）
- **问题**：三栏布局中每栏都有独立的 WorkspacePanel 包裹
- **具体表现**：
  - 左侧案例列表：独立 WorkspacePanel + SectionHeader
  - 中间证据区：WorkspacePanel + SectionHeader
  - 右侧副驾区：WorkspacePanel + SectionHeader

## 优化策略

### 统一原则
1. **减少边框** - 移除不必要的 border 和分隔线
2. **移除冗余标题** - 删除功能区块的 title + description
3. **简化结构** - 减少嵌套的 WorkspacePanel
4. **视觉统一** - 使用一致的间距和间距系统
5. **留白呼吸** - 用间距而非边框来区分内容

---

## 实施步骤

### 第一步：简化 Center 页面
1. 移除 ContextActionStrip 的标题和描述文字
2. 移除视频墙区域的 SectionHeader（"统一视频墙"）
3. 合并侧边栏列表区域到主区域，移除独立边框
4. 移除右侧详情区的 SectionHeader
5. 简化"关联告警"、"关联资料"等区块的标题
6. 减少 StatusPill 的使用，用更简洁的展示方式

### 第二步：简化 Gallery 页面
1. 移除 PageHeader 的 description
2. 移除 SectionBlock 的 title 和 description（保留空标签）
3. 右侧三个 SectionBlock 合并为一个简洁面板
4. 简化图片卡片的边框和背景

### 第三步：简化 Media 页面
1. 移除 ContextActionStrip 的 summary 文字
2. 移除模式切换区域的 WorkspacePanel 包裹
3. 移除每个 WorkspacePanel 的 SectionHeader
4. 三栏布局改为更简洁的展示方式

### 第四步：统一组件样式
1. 调整全局间距（从 gap-4 调整为 gap-6 或更大）
2. 简化 StatusPill 样式
3. 统一卡片圆角大小

---

## 预期效果
- 页面更加通透，不再有密集的边框
- 视觉重点放在核心内容上
- 整体风格简约、专业、有呼吸感
- 保持功能完整性的同时提升视觉体验
