# 巡检宝 - 视觉层级规范

## 版本
v1.0 - 2026-04-07

## 核心理念
专业、简洁、高效、耐疲劳

---

## 视觉层级定义

### Level 0 - 全局层（固定不变）
**组件**: 侧边栏、顶栏
**特点**: 
- 固定位置，不随内容滚动
- 最高层级（z-index: 1200+）
- 视觉权重：中性/次要

**样式规范**:
```css
.sidebar, .header {
  position: fixed;
  z-index: var(--z-fixed);
  background: rgb(var(--canvas));
  border: none;
}
```

---

### Level 1 - 容器层（页面骨架）
**组件**: WorkspacePanel, MainPanel, Card
**特点**:
- 页面主要内容区域
- 统一的圆角（28px）和内边距
- 视觉权重：主要背景

**样式规范**:
```css
.workspace-panel {
  border-radius: var(--radius-2xl); /* 28px */
  background: rgb(var(--surface));
  border: 1px solid rgb(var(--border));
  padding: var(--space-4); /* 16px */
}

@media (min-width: 768px) {
  .workspace-panel {
    padding: var(--space-5); /* 20px */
  }
}
```

---

### Level 2 - 区块层（功能区块）
**组件**: SectionHeader, MetricRow, ActionBar
**特点**:
- 页面内的功能区块
- 视觉层级：标题 > 描述
- 交互层级：可点击区域

**样式规范**:
```css
.section-header {
  display: flex;
  align-items: flex-start;
  justify-content: space-between;
  gap: var(--space-3);
}

.section-title {
  font-size: var(--text-lg); /* 16px */
  font-weight: 600;
  letter-spacing: -0.03em;
  color: rgb(var(--text-primary));
}

.section-eyebrow {
  font-size: var(--text-xs); /* 11px */
  font-weight: 600;
  letter-spacing: 0.16em;
  text-transform: uppercase;
  color: rgb(var(--text-tertiary));
}

.section-description {
  font-size: var(--text-sm); /* 12px */
  line-height: 1.75;
  color: rgb(var(--text-secondary));
}
```

### Level 4 - 交互层（用户操作）
**组件**: Button, Input, Select, Switch, Checkbox
**特点**:
- 用户可以直接交互
- 状态反馈明确（hover、active、disabled）
- 视觉权重：主要操作 > 次要操作

**样式规范**:

#### Button（按钮）
```css
.btn {
  border-radius: var(--radius-lg); /* 12px */
  transition: all var(--duration-fast) var(--ease-out);
  cursor: pointer;
}

/* 尺寸 */
.btn-sm {
  padding: var(--space-2) var(--space-3); /* 8px 12px */
  font-size: var(--text-xs);
}

.btn-md {
  padding: var(--space-2) var(--space-4); /* 8px 16px */
  font-size: var(--text-sm);
}

.btn-lg {
  padding: var(--space-3) var(--space-5); /* 12px 20px */
  font-size: var(--text-base);
}

/* 变体 */
.btn-primary {
  background: rgb(var(--accent));
  color: white;
}

.btn-primary:hover {
  background: rgb(var(--accent-soft));
  transform: translateY(-1px);
  box-shadow: var(--shadow-hover);
}

.btn-primary:active {
  transform: scale(0.98);
}

.btn-secondary {
  background: rgb(var(--surface-raised));
  border: 1px solid rgb(var(--border));
  color: rgb(var(--text-primary));
}

.btn-secondary:hover {
  background: rgb(var(--surface-elevated));
  border-color: rgb(var(--border-hover));
}

.btn-ghost {
  background: transparent;
  color: rgb(var(--text-secondary));
}

.btn-ghost:hover {
  background: rgb(var(--surface-raised));
  color: rgb(var(--text-primary));
}
```

#### Input（输入框）
```css
.input {
  border-radius: var(--radius-md); /* 8px */
  border: 1px solid rgb(var(--border));
  background: rgb(var(--surface-raised));
  transition: all var(--duration-fast) var(--ease-out);
}

.input:hover {
  border-color: rgb(var(--border-hover));
}

.input:focus {
  border-color: rgb(var(--accent));
  box-shadow: 0 0 0 3px rgb(var(--accent) / 0.10);
}
```

## 状态反馈规范

### 统一状态定义

#### Hover（悬浮）
```css
/* 所有交互元素的悬浮状态 */
*:hover {
  transition: all var(--duration-fast) var(--ease-out);
}

/* 按钮悬浮 */
.btn:hover {
  transform: translateY(-1px);
  box-shadow: var(--shadow-hover);
}

/* 卡片悬浮 */
.card:hover {
  border-color: rgb(var(--border-hover));
  box-shadow: var(--shadow-hover);
}

/* 列表项悬浮 */
.list-item:hover {
  background: rgb(var(--surface-raised));
}
```

#### Active（激活）
```css
.btn:active {
  transform: scale(0.98);
}

.list-item:active {
  background: rgb(var(--accent) / 0.08);
}
```

#### Disabled（禁用）
```css
*:disabled {
  opacity: 0.5;
  cursor: not-allowed;
  pointer-events: none;
}
```

#### Loading（加载中）
```css
.loading {
  position: relative;
  pointer-events: none;
}

.loading::after {
  content: '';
  position: absolute;
  inset: 0;
  background: rgb(var(--surface) / 0.8);
  display: flex;
  align-items: center;
  justify-content: center;
}
```

## 动画规范

### 统一动画时长
```css
--duration-fast: 150ms;   /* 微交互 */
--duration-normal: 200ms;  /* 标准过渡 */
--duration-slow: 300ms;   /* 大型动画 */
```

### 统一缓动函数
```css
--ease-out: cubic-bezier(0.4, 0, 0.2, 1);     /* 标准缓出 */
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1); /* 弹性效果 */
```

### 关键动画

#### 页面进入动画
```css
@keyframes enter-up {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.page-enter {
  animation: enter-up 200ms var(--ease-out) forwards;
}

/* 列表项 stagger 效果 */
.list-item:nth-child(1) { animation-delay: 0ms; }
.list-item:nth-child(2) { animation-delay: 50ms; }
.list-item:nth-child(3) { animation-delay: 100ms; }
/* ... */
```

#### 按钮点击反馈
```css
@keyframes click-feedback {
  0% { transform: scale(1); }
  50% { transform: scale(0.98); }
  100% { transform: scale(1); }
}

.btn:active {
  animation: click-feedback 100ms var(--ease-out);
}
```

#### 加载状态
```css
@keyframes shimmer {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}

.skeleton {
  background: linear-gradient(
    90deg,
    rgb(var(--surface-raised)) 25%,
    rgb(var(--surface-elevated)) 50%,
    rgb(var(--surface-raised)) 75%
  );
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}
```

## 响应式断点

```css
/* 移动端 < 768px */
@media (max-width: 767px) {
  .workspace-panel {
    padding: var(--space-3);
    border-radius: var(--radius-xl);
  }
}

/* 平板端 768px - 1279px */
@media (min-width: 768px) and (max-width: 1279px) {
  .workspace-panel {
    padding: var(--space-4);
    border-radius: var(--radius-2xl);
  }
}

/* 桌面端 >= 1280px */
@media (min-width: 1280px) {
  .workspace-panel {
    padding: var(--space-5);
    border-radius: var(--radius-2xl);
  }
}
```

## 优先级权重

### P0 - 核心（必须一眼看到）
**组件**: 监控状态指示、告警数量徽章
**视觉权重**: 
- 颜色：强调色（accent）
- 尺寸：较大
- 动画：带脉冲效果

### P1 - 常用（高频使用）
**组件**: 媒体库、智能协同入口
**视觉权重**:
- 颜色：主色调
- 尺寸：标准
- 动画：标准

### P2 - 辅助（低频使用）
**组件**: 系统管理、设置
**视觉权重**:
- 颜色：淡化（text-tertiary）
- 尺寸：较小
- 动画：简化

---

## 成功指标

1. **一致性**: 所有组件使用统一的圆角、间距、阴影
2. **可预测性**: 相同类型的交互有相同的视觉反馈
3. **响应性**: 所有动画流畅，无明显卡顿
4. **可访问性**: 满足 WCAG 2.1 AA 标准
